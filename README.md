# mac_emacs
mac环境下的emacs配置

将目录名mac_emacs改为.emacs.d, 放在~目录下即可使用

# elpy安装的package
- exec-path-from-shell
- elpy
- color-theme-sanityinc-tomorrow
- auto-complete hideshowvis
- mark-multiple
- autopair
- paredit
- ace-jump-mode
- wrap-region
- undo-tree
- highlight-symbol
- expand-region
- smex
- window-number
- yasnippet
- tabbar

# mode
## python mode

    pip install rope
    pip install jedi
